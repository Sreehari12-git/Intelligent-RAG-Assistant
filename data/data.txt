Node.js is a JavaScript runtime built on Chrome's V8 engine.

Express.js is a minimal and flexible Node.js web application framework.
It provides a robust set of features for building web and mobile applications.
You can create routes, handle requests, and build REST APIs with Express.js.

Example:
const express = require('express');
const app = express();
app.get('/', (req, res) => res.send('Hello World'));
app.listen(3000);

React is a frontend JavaScript library for building user interfaces.
